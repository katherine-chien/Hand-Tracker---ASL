# Hand-Tracker---ASL
Over 70 million people use sign language globally. At 500,000, American Sign Language, ASL, is the most used sign language in the world. Within ASL, about 12% to 35% is fingerspelling. Finger-spelling is a form of sign language where one would use one hand to sign one letter at a time to spell out words.

Pain:
Even with an abundance of people using sign
language, there is still a huge barrier between
those who are hard of hearing and those who
are not.

Need:
A tool to help break down the barrier to make
using sign language more convenient, also
providing those using sign language with more
equal opportunities.


Detailed description of the top solution：
A program is written to recognize the movement of the hand and to analyze the gesture of the hand. Then, the program would examine which
fingerspelling hand sign the gesture is closest to. Finally, it would output the letter that corresponds to the said sign for those that does not understand sign language to read.
